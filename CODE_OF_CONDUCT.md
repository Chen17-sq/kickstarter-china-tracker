# Code of Conduct

## 三条原则

1. **对项目而非对人**。批评代码、数据、设计选择都欢迎，针对人不行。
2. **数据要有来源**。补品牌、补一句话、报错都附链接 / 截图 / commit hash，方便复核。
3. **不要消耗别人的时间**。issue 写清现象 vs 预期；PR 写清改了什么、为什么。

## 不允许

- 仇恨言论 / 人身攻击
- 政治 / 民族主义争吵（这是个数据 tracker，不是论坛）
- 发广告 / 拉新
- 提交未经核实的"中国品牌"想刷曝光（low-source PR 直接 close）

## 不喜欢这个项目

不必维护，fork 走自己改。MIT 给了你这个权利。
